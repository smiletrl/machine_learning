## 📂 目录导航

为了将“硬核理论”与“工程实战”分离，项目结构如下：

- [**`docs/README.md`**](./docs/README.md) -> **【强烈推荐】** 极致舒适的数学推导文档。包含**核心符号速查表**与反向传播的**两大通用法则**。
- [**`code/`**](./code/) -> 代码实战区。包含纯 NumPy 的梯度推导验证、Mini-batch 矩阵加速训练脚本，以及工业级框架对照实现。
