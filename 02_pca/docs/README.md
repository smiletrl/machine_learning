# 主成分分析 (PCA) 纯代数推导过程

主成分分析（PCA）的本质，是寻找一个新的坐标系，使得原始数据投影到这个新坐标系后，**各个维度之间不再相关（协方差为 0）**，从而方便我们提取主要信息（方差最大的维度）。

以下是完全基于矩阵代数的推导过程。

## 1. 数据准备与去中心化

假设我们有 $n$ 个样本，每个样本有 $d$ 个维度。

* 原始样本表示为列向量：$x_1, x_2, \dots, x_n$ 。

**第一步：计算均值向量** (计算均值不受无偏估计影响，分母仍为 $n$)


$$x_m = \frac{1}{n} \sum_{i=1}^n x_i$$

**第二步：计算均值偏差（去中心化）**
我们将每个样本减去均值，得到去中心化后的向量 $B_i$ ：


$$B_i = x_i - x_m$$


将所有去中心化后的列向量拼成一个 $d \times n$ 的偏差矩阵 $B$：


$$B = [B_1, B_2, \dots, B_n]$$

## 2. 计算原始无偏协方差矩阵

根据无偏协方差矩阵的定义，去中心化数据矩阵 $B$ 的协方差矩阵 $S$ 为：


$$S = \frac{1}{n-1} B B^T$$

$S$ 是一个 $d \times d$ 的实对称矩阵。实对称矩阵有一个非常优美的数学性质：**它一定可以被正交矩阵对角化**。

即存在一个标准正交矩阵 $P$（满足 $P^T = P^{-1}$）和一个对角矩阵 $D$ ，使得：


$$S = P D P^T$$


其中：

* $P$ 的每一列都是 $S$ 的单位特征向量。
* $D$ 是对角矩阵，对角线上的元素 $\lambda_1, \lambda_2, \dots, \lambda_d$ 是对应的特征值（通常按从大到小排列）。

通过矩阵代数变形，我们可以得到 $D$ 的表达式：


$$D = P^T S P$$

## 3. 坐标变换（投影）

我们定义 $P^T$ 为投影矩阵。将原始去中心化数据 $B$ 乘以投影矩阵 $P^T$，就得到了新坐标系下的数据矩阵 $Y$：


$$Y = P^T B$$


$Y$ 也是一个 $d \times n$ 的矩阵，它的每一列就是原始样本在新坐标系下的坐标。

## 4. 核心证明：为什么这个投影是有效的？

我们只需要计算一下新数据 $Y$ 的无偏协方差矩阵（记作 $S_Y$），看看它变成了什么样。

根据无偏协方差的定义：


$$S_Y = \frac{1}{n-1} Y Y^T$$

将坐标变换公式 $Y = P^T B$ 代入进去：


$$S_Y = \frac{1}{n-1} (P^T B)(P^T B)^T$$

利用矩阵转置的性质 $(AB)^T = B^T A^T$，将括号展开：


$$S_Y = \frac{1}{n-1} (P^T B)(B^T P)$$

利用矩阵乘法的结合律，把 $P^T$ 和 $P$ 提取到两边：


$$S_Y = P^T \left( \frac{1}{n-1} B B^T \right) P$$

注意到括号里的 $\frac{1}{n-1} B B^T$ 正好就是我们前面定义的**原始无偏协方差矩阵** $S$：


$$S_Y = P^T S P$$

根据我们在第 2 步中得出的实对称矩阵对角化结论（$D = P^T S P$），我们得到了最终的绝妙结果：


$$S_Y = D = \begin{bmatrix} \lambda_1 & 0 & \dots & 0 \\ 0 & \lambda_2 & \dots & 0 \\ \vdots & \vdots & \ddots & \vdots \\ 0 & 0 & \dots & \lambda_d \end{bmatrix}$$

## 5. 结论与降维操作

通过上面的推导，我们发现新数据 $Y$ 的无偏协方差矩阵 $S_Y$ 变成了一个**对角矩阵**。这意味着：

1. **彻底去相关：** 矩阵非对角线元素全为 0，说明新坐标系下各个特征之间完全独立，没有任何冗余信息。
2. **方差即特征值：** 矩阵对角线上的元素 $\lambda_1, \lambda_2, \dots$ 就是新特征的无偏样本方差。方差越大，包含的信息越多。

**如何降维？**
因为特征值 $\lambda$ 已经按从大到小排列，如果我们要将数据从 $d$ 维降到 $k$ 维（$k < d$），只需要：

1. 取对角矩阵 $D$ 前 $k$ 个最大的特征值。
2. 取正交矩阵 $P$ 的前 $k$ 个特征向量，转置后作为新的降维投影矩阵 $P_{k}^T$ 。
3. 计算降维后的数据：$Y_{k} = P_{k}^T B$ 。

---

### 补充证明：为什么投影后的矩阵 $Y$ 也是中心化的？

**已知条件：**

1. 原始偏差矩阵 $B = [\tilde{x}_1, \tilde{x}_2, \dots, \tilde{x}_n]$ 是已经中心化的，这意味着它所有列向量（样本）的均值向量为 $\mathbf{0}$ ：

$$\mu_B = \frac{1}{n} \sum_{i=1}^n \tilde{x}_i = \mathbf{0}$$


2. 新数据矩阵 $Y = P^T B$。如果我们将 $Y$ 按列展开，记为 $Y = [y_1, y_2, \dots, y_n]$ ，那么每一个新样本向量的计算公式为：

$$y_i = P^T \tilde{x}_i$$



**证明目标：**
证明新数据集 $Y$ 的均值向量 $\mu_Y$ 同样为 $\mathbf{0}$ 。

**推导过程：**
根据均值的定义，新数据集 $Y$ 的均值向量 $\mu_Y$ 表达式为：


$$\mu_Y = \frac{1}{n} \sum_{i=1}^n y_i$$

将 $y_i = P^T \tilde{x}_i$ 代入上式中：


$$\mu_Y = \frac{1}{n} \sum_{i=1}^n \left( P^T \tilde{x}_i \right)$$

因为投影矩阵 $P^T$ 是一个常数矩阵，它不随样本下标 $i$ 的改变而改变，根据矩阵乘法的分配律，我们可以把 $P^T$ 提到求和符号 $\sum$ 的外面：


$$\mu_Y = P^T \left( \frac{1}{n} \sum_{i=1}^n \tilde{x}_i \right)$$

此时，括号内部的 $\frac{1}{n} \sum_{i=1}^n \tilde{x}_i$ 恰好就是原始数据矩阵 $B$ 的均值向量 $\mu_B$ 。我们将其代入：


$$\mu_Y = P^T \cdot \mu_B$$

因为原始数据已经过中心化处理，即 $\mu_B = \mathbf{0}$，所以：


$$\mu_Y = P^T \cdot \mathbf{0} = \mathbf{0}$$

**结论：**
新矩阵 $Y$ 的各个维度均值完全为 $\mathbf{0}$。这证明了**对中心化后的数据进行线性投影，得到的新数据依然是中心化的**。

因此，在后续计算 $Y$ 的无偏协方差矩阵时，无需再单独减去均值，直接使用 $S_Y = \frac{1}{n-1} Y Y^T$ 在数学上是完全合法且严谨的。
